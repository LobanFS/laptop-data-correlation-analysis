# Laptop-data-correlation-analysis
Этот проект посвящён корреляционному анализу характеристик ноутбуков и исследованию факторов, влияющих на их стоимость.
# Цель:
Изучить зависимости между характеристиками ноутбуков и их стоимостью с помощью методов корреляционного и статистического анализа, а также проверить статистические гипотезы о распределении и взаимосвязях признаков.
# Задачи:

## 1. Проверка распределения на нормальность
Проанализировать распределение значений 5 метрических переменных.
Сформулировать нулевую и альтернативную гипотезы:
H₀: распределение не отличается от нормального; H₁: распределение отличается.
Провести тесты нормальности (например, Shapiro–Wilk, Kolmogorov–Smirnov, D’Agostino) для выбранных признаков.
Построить гистограммы распределения с наложенной кривой нормального распределения и интерпретировать результаты.

## 2. Тест Хи-квадрат
Выбрать не менее двух пар категориальных переменных.
Сформулировать гипотезы независимости признаков:
H₀: переменные независимы; H₁: между переменными существует зависимость.
Построить таблицы сопряжённости (contingency tables) и вычислить статистику χ².
Определить p-value и сделать вывод о статистической значимости.
Визуализировать зависимости (гистограммы, stacked bar chart, heatmap).

## 3. Парная корреляция
Исследовать взаимосвязь не менее трёх пар количественных переменных.
Заполнить итоговую таблицу с указанием силы, направления и значимости корреляций.
Визуализировать зависимости (scatter plot, regression plot, heatmap).

## 4. Частная корреляция
Рассчитать парный коэффициент корреляции между двумя переменными.
Рассчитать частный коэффициент корреляции, исключив влияние третьей переменной.
Сравнить результаты и сделать вывод о реальной силе взаимосвязи.
Визуализировать результаты (scatter plot с трендом и контрольной переменной).
